教育部副部长李卫红来校调研

作者: 陈伟 / 来源:党委宣传部 / 时间:2013-11-28 / 点击: 5462 / 评论: 0 /

11月27日，教育部副部长李卫红来校调研，校党委书记王志强、校长李言荣、副校长余敏明等校领导及学校办等有关部门负责人陪同调研。

王志强代表学校欢迎李卫红副部长来校调研，感谢教育部长期以来对电子科技大学改革发展等各项事业的关心和支持。他向李卫红汇报了学校的人才培养、科学研究、学科建设、师资队伍建设、新校区建设、校园文化建设等工作，介绍了我校的建校历史、发展规划以及目前正在开展的党的群众路线教育实践活动和学习贯彻落实党的十八届三中全会精神等情况。李言荣作了补充汇报。

李卫红对学校近年来各项事业的发展给予充分肯定。她说，电子科技大学在我国高等教育发展中发挥着重要作用，为我国国防现代化建设和经济社会发展做出了重要贡献，是一所值得国家重视和投入的大学。她说，新一届校领导班子团结带领全校师生员工开拓进取、抢抓机遇，继承和发扬了“求实求真、大气大为”的成电精神，整个学校继续保持了良好的发展势头，希望全体师生员工扎扎实实开展好党的群众路线教育实践活动，学习好、贯彻好、落实好党的十八届三中全会精神，借鉴国内外一流大学办学经验，为我国高等教育发展再立新功，为中华民族的伟大复兴贡献力量。

李卫红在调研中还详细了解了我校师生员工学习、生活、工作等情况。她在参观清水河校园后，对我校新校区的规划、建筑、绿化等予以充分肯定。
